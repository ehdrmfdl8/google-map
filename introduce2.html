미아찾기 솔루션

우리는 실종된 미아를 보다 효율적으로 찾기 위해서 어린아이나 장애인에게 LoRa 모듈을 착용함으로써 실종사고를 예방할수가 있다.
우리팀의 미아 찾기 솔루션은 키워드는 3가지인데 Google Api,Gps 모듈,드론이 있다. 그림에서 보듯이
1.관리자로 부터 위치데이터를 요청하는것
2.미아(LoRa모듈)로부터 위치데이터를 받아오는것
3.관리자는 프로그램을 이용하여 지도를 통해 드론에게 해당하는 위치로 이동명령을 하는것
4.드론은 GPS데이터를 토대로 미아의 위치로 이동하여 실시간으로 영상을 관리자에게 보여주는것
LoRa를 이용하여 1대 다 통신하여 관리자가 원하는 미아의 위치를 선별적으로 확인할수 있는 프로그램이다.

What's Google API

우리는 GPS데이터를 토대로 Google Map으로 표현하기 위해서 Google API를 이용하였는데 위의 그림처럼 Google Map을 얻기위해
사용자가 Google에서 인증을 받아 API Key를 발급 받고 javascript내에 발급받은 API키를 입력해주면 인터넷이 연결된 경우에
Google에서 사용자에게 Google Map(Google Service)을 제공해주는 방식이다.

What's LoRa Module

LoRa는 Long-Range의 약자로 최대 15km까지의 무선 통신이 가능한 플랫폼이다. 우리는 GPS를 통해 사용자가 가지고 있는 모듈의 위치 정보를 파악한다.
이를 아두이노 프로 미니를 통해 GPS의 데이터를 모듈 내에서 처리를 하고, LoRa를 통해 Main PC로 전송하여 위치 데이터를 통해 모듈의 위치를 파악한다.
모듈은 배터리를 통해 동작을 하고, 배터리 충전 회로가 내장되어 있어 충전이 가능하다.
외부 프레임의 경우, 회로의 삽입과 수리가 용이하도록 조립식으로 구성되어 있고, 크기는 가로 7cm, 세로 5.5cm, 높이 5.5cm로 비교적 작고 가벼운 형태
이다, 실제 동작 실험시 2km까지 신호가 확인이 되었으며, 이는 드론이 비행 가능한 거리, 1.5km를 상회하는 거리이므로 신호 전송에
무리가 없음을 확인하였다.
